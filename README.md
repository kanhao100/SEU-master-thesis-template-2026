## 本模板修改内容

- 对于每一章第一页、摘要、致谢、作者简介的页眉进行补全
- 原模板适用于双面打印的形式，提供去掉空白页的选项，print:印刷版，全书强制插空白页, digital: 电子版，仅正文强制插空白页
- 修复学硕模板中A4小封面，缺少研究方向字样；日期被挤到下一页等问题；
